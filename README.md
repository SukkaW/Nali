# Nali

## Installation

```bash
$ ./configure
$ make
# make install
```


## Usage

```
$ nali [IP] # Get IP Geolocation Informatiuon
$ nali-update # Update QQWry.Dat From QQWry Mirror
$ nali-ping
$ nali-dig
$ nali-traceroute
$ nali-tracepath
$ nali-dig
$ nali-nslookup
```
